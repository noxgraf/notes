SMB

  • NetExec

TCP: 139, 445

nmap -sV -sC -p139,445 -vv <target>

nmap -sS -vv --scripts=smb-vuln* <target>

Check guest / no pass​

Maybe there's a public share with good loot.

smbclient -L //<target> -U "guest"

smbclient -L //<target> -N

Connect​

If you find a public share or have a valid credential.

  • ls list
  • cd change dir
  • less and more read files from shell
  • put uploads files
smbclient //<target>/<share> -U 'sherlock' -P 'i@mth3best'

Mount​

Good to combo with grep -r to search for passwords and regex.

mount -t cifs "//<target>/<remote_share>/" /mnt/<local_share>

Check if SMB Relay is possible​

If you see Message signing enabled but not required or disabled, its vulnerable.

nmap --script smb2-security-mode -p 445 <target>

smbmap​

Auto-tool.

smbmap -H <target> -u 'j.watson' -p 'h0lms!smybestie' -s <share>
