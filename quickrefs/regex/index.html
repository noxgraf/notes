Regex

Regular Expressions check a series of chars for pattern matches.

/REGEX/

W3 Cheat sheet

Flags​

// global
/word/g
// global no case
/word/gi

Metachars​

// digits (0-9)
\d
// word char (alphanum and _) || \W non-word
\w
// whitespace (space, tabs, newline) || \S non-space
\s
// tab only
\t
// backspace
[\b]
// carriage returns
\r
// new lines
\n

Special Chars​

// escape
\
// wildcard
.
// zero or more
*
// one or more
+
// charset
[]
// negate charset
[^]
// optional  (can apear one time or not appear at all)
?
// or
|
// separate entity
( )

Assertions​

// beginning
^
// end
$
// beginning or end
\b
// not beginning or end
\B

Charsets, Ranges and Repetition​

// robsu
// first match
/robco/
// global
/robco/g
// charset -> robco, lobco
/[rl]obco/g
// charset exclude -> robco, bobco, cobco, !lobco
/[^l]obco/g
// charset range
/[a-z]obco/g
// charset range, uppercase
/[A-Z]obco/g
// charset range, both
/[A-Za-z]obco/g
// charset range, numbers
/[0-5]obco/g

// match only 5 letters string
/^[a-z]$/g

// match ors
/(pet | space | toy) rabbit/g

// charset 4 numbers in a row
/[0-9][0-9][0-9][0-9]/g
// same, but smarter
/[0-9]{4}/g
// charset lowercase 5 chars
/[a-z]{5}/g
// charset lowercase min 5 chars, max 10 chars
/[a-z]{5,10}/g
// charset lowercase min 5 chars, max infinite
/[a-z]{5,}/g
// infinite
/[0-9]+/g
