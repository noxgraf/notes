Basics

  • Local User and Group Management: lusrmgr.msc. Provides insight on machine local users and permission groups.
  • Computer Management: compmgmt. System Tools, Storage, and Services and Applications.
  • System Information: msinfo32. Complete info about the system in a structured manner. Can find all environment variables here.
  • Resource Monitor: resmon. Buffed task manager. Helps with troubleshoot. Used this to create private online sessions in GTA V, suspending the process for about 15s.
  • Registry: regedit. Central hierarchical database used to store information necessary to configure the system for one or more users, applications, and hardware devices.

Shell History: C:\Users\User\AppData\Roaming\Microsoft\Windows\PowerShell\PSReadLine\ConsoleHost_History

Processes and Services​

BINARY_PATH_NAME path to associated executable.
SERVICE_START_NAME account used to run the service.

  • All of the services configurations are stored on the registry under HKLM\SYSTEM\CurrentControlSet\Services\.
  • If the executable associated with a service has weak permissions that allow an attacker to modify or replace it, the attacker can gain the privileges of the service's account trivially.
# list services
wmic service get name,pathname,startname | findstr /i "user"

# list processes
tasklist /FI "<field> eq <string>" 

tasklist /V /fi "username eq system"

# list all
sc query

sc query state= all | find "<keyword>"

sc queryex type=service

# check specific
sc qc <service>

# start / stop services

sc start <name>

Password Spray​

  • [+]: A successful authentication.
  • [-] STATUS_LOGON_FAILURE: The password is incorrect for this account.
  • [-] STATUS_ACCOUNT_DISABLED: The account exists but has been disabled.
  • [-] STATUS_ACCOUNT_LOCKED_OUT: The account has been locked out.
    • If you see this, stop spraying immediately, you may be triggering lockouts.
  • (Pwn3d!): If this appears after a successful login, the account has administrator privileges on the target host. This is an even bigger win than a standard domain user.
nxc smb <dc_ip> -u clean_users.txt -p '<password>' --continue-on-success  --jitter 2-5

Powershell​

  • It's a command dump for now. If I use something here often, I'll create a new heading for the specific command.
# list commands
Get-Command 

# get command by type (like cmdlet)
Get-Command -CommandType "Function" 

# get help on specific command, like man
Get-Help <command> 

# get a list of alias, like cd is Get-Content
Get-Alias 

# cd, don't need -path
Set-Location -Path 

# ls / dir
Get-ChildItem 

# cat
Get-Content 

# touch file or make diretory
New-Item -Path "<path>" -ItemType "Directory" 

# remove directory or file
Remove-Item -Path "<full_path>" 

# copy
Copy-Item -Path "<path>" -Destination "<new_path>" 

# ls + pipe with sort by length
Get-ChildItem | Sort-Object Length 

# filter by extension
Get-ChildItem | Where-Object -Property "Extension" -eq ".txt"  

# filter by name
Get-ChildItem | Where-Object -Property "Name" -like "flag*"  

# ls only columns you want
Get-ChildItem | Select-Object Name,Length 

# grep
Select-String -Path "<file>.txt" -Pattern "password" 

# services
Get-WmiObjectWin32_Service | select Name, Startname

# systeminfo
Get-ComputerInfo 

# local user accounts
Get-LocalUser 

# ipconfig
Get-NetIPConfiguration 

# ipconfig 2.0
Get-NetIPAddress 

# list running processes
Get-Process

# list running services 
Get-Service 

# looks like netstat
Get-NetTCPConnection 

# get filehash for authenticity checks
Get-FileHash -Path <file> 

# Invoke-Command can execute payload remotely on other machines >:D
Get-Help Invoke-Command -examples 

# ignore restrictions
powershell.exe -ExecutionPolicy Bypass .\script.ps1 

# allow scripts
powershell.exe -ExecutionPolicy Unrestricted .\script.ps1 

# execute base64 encoded cmd
powershell.exe -EncodedCommand <encoded> 

# hide CLI
powershell.exe -WindowStyle Hidden 

# search cmdlet in online repo
Find-Module -Name "PowerShell*"  

#  download cmdlet from online repo
Install-Module -Name "PowerShellGet" 

# list sofware and its versions
wmic product get name,version,vendor
