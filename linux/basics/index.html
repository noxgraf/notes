Basics

/etc -> files used by the system
/var -> (variable data) frequently accessed or written by services and applications
/var/logs -> logs
/root -> root system user
/home -> users directories
/tmp -> temporary files

cron: a time-based job scheduler.
crontab: special file with formatting recognized by cron process.

  • Crontab generator
  • Crontab guru

crontab example:

# Runs backup of Documents, * is a wildcard (crontab -e nvim)

0 */12 * * * cp -R /home/moriarty/Documents /var/backups/

File Descriptors are handlers that the system uses to make communication between processes and other resources like pipe. When you have a python script that opens a text file, the kernel points a fd of that process to act as a bridge between the text file and the script.

  • 0 | Standard IN
  • 1 | Standard OUT
  • 2 | Standard Error

find​

  • -mtime -10 - modified last 10 days
  • -atime -10 - accessed last 10 days
  • -cmin -60 - changed last hour
  • -amin -60 - accessed last hour
  • -size +50M - files at least 50MB
find <place> -name <string> -type <type> -perm <permlvl>

# find file by name and redirect errors
find <location> -name <file_name> 2>/dev/null

# find bad perms
find / -type f -perm 0777 2>/dev/null

# find executables
find / -perm -a=x 2>/dev/null

# find files for user
find / -user <user> 2>/dev/null

# find writable folders
find / -writable -type d 2>/dev/null
find / -perm -222 -type d 2>/dev/null
find / -perm -o w -type d 2>/dev/null

# world-executable folders
find / -perm -o x -type d 2>/dev/null

# SUID (4000) -> u+s & SGID (2000) -> g+s | sticky (1000) -> +t
find / -perm -u=s -type f 2>/dev/null
find / -type f -perm -04000 -ls 2>/dev/null

Network​

# network legacy
ifconfig 
netstat

# network
ip a s

# network routes | route
ip r

# arp table | arp -a 
ip n

# sockets statistics --udp --tcp --listen --process --numeric
ss -tupln

Processes and Services​

# processes | ps axjf
ps aux

# real time processes
top

# kill process | pkill -u <user>
kill <PID> 

# start, stop, enable, disable, status
systemctl <option> <service>

System Info​

# -s -> kernel name | -k -> kernel release | -v -> kernel version | -o -> OS
uname -a

# files with infos
cat /proc/version
cat /etc/os-release
cat /etc/issue

Environment Variables​

# environment variables
env

# list PATH
echo $PATH

# set a variable
export VAR=value

# unset a variable
unset VAR

terminal basics​

  • This is too basic, but sometimes you get nervous.
# user privilege and group memberships
id <user>

# shell history
history

# change dir
cd 

# read content
cat file.txt
less file.txt
more file.txt
head file.txt

# list dir
ls -la 

# list partitions
df -h

# set alias | unalias
alias ns="ls /usr/share/nmap/scripts/ | grep"

# set function | unset -f nscript
nscript() { ls /usr/share/nmap/scripts/ | grep "$@" } 

# start in background | CTRL + Z
<command>& 

# focus back
fg 

# query installed packages and versions | debian
dpkg -l
